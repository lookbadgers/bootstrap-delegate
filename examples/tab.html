Create a next / previous button for use with bootstrap tabs with minimal additional code.

<button type="button" class="btn btn-primary" data-delegate="tab-previous" data-target="#my-tabs">Previous Tab</button>
<button type="button" class="btn btn-primary" data-delegate="tab-next" data-target="#my-tabs">Next Tab</button>
